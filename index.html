Getting Started with Canvas

Setting up canvas

							const canvas = document.getElementById('myCanvas');
							const ctx = canvas.getContext('2d');
							ctx.canvas; // HTMLCanvasElement;

Drawing a Rectangle

							
	// Draw a filled rectangle
	// fillRect(x, y, width, height)
	ctx.fillStyle = 'green';
	ctx.rect(50, 50, 100, 75);
	ctx.fillRect(50, 50, 100, 75);

	// Draw a rectangle with border
	ctx.strokeRect(150, 150, 50, 50);

	// Sets the pixels in an area to transparent black rgba(0,0,0,0)
	ctx.clearRect(50, 50, 30, 30);

	// Draw rounded rectangle
	roundRect(10, 20, 150, 100, [40]);

Drawing a Line

											// Draw a line
				// moveTo(x, y), lineTo(x, y)
				// ctx.lineCap = "butt" (def) | "round" | "square";
				// ctx.lineJoin = "miter" (def) | "round", "bevel";
				ctx.lineWidth = 10;
				ctx.beginPath();
				ctx.moveTo(50, 140);
				ctx.lineTo(150, 60);
				ctx.lineTo(250, 140);
				ctx.closePath();
				ctx.stroke();

				// Draw a curve line
				// bezierCurveTo() || quadraticCurveTo()
				ctx.bezierCurveTo(50, 50, 90, 10, 100, 50);

Drawing Arcs & Circles

							//Draw a circle
//ctx.arc(x, y, radius, startAngle, endAngle, anticlockwise);
ctx.beginPath();
ctx.arc(75, 75, 50, 0, 2 * Math.PI);
ctx.stroke();

//Draw a circular arc
//arcTo(x1, y1, x2, y2, radius)
ctx.arcTo(100, 100, 300, 150, 50);

//Draw an ellipse
//ellipse(x, y, radiusX, radiusY, rotation, startAng, endAng, counterclockwise)
ctx.ellipse(100, 100, 50, 75, Math.PI / 4, 0, 2 * Math.PI);

Making clips

							
					// Draw clip
					ctx.beginPath();
					ctx.arc(100, 75, 50, 0, Math.PI * 2);
					ctx.clip();

					// Draw stuff that gets clipped
					ctx.fillStyle = "blue";
					ctx.fillRect(0, 0, canvas.width, canvas.height);

		// Draw radial gradient
		// createRadialGradient(x0, y0, r0, x1, y1, r1)

Draw a text string

							
		// fillText(text, x, y, maxWidth)

Canvas Animation

Basic animation steps to draw a frame:

  1. Clear the canvas Unless the shapes you'll be drawing fill the complete canvas (for instance a backdrop image), you need to clear any shapes that have been drawn previously. The easiest way to do this is using the clearRect() method.
  2. Save the canvas state If you're changing any setting (such as styles, transformations, etc.) which affect the canvas state and you want to make sure the original state is used each time a frame is drawn, you need to save that original state.
  3. Draw animated shapes The step where you do the actual frame rendering.
  4. Restore the canvas state If you've saved the state, restore it before drawing a new frame.
